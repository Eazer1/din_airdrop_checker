# DIN (WEB3GO) Airdrop Checker
## Настройка и запуск:
- `pip install -r requirements.txt`
- В `proxies.txt` построчно вставляем прокси (любой формат)
- В `wallets.txt` построчно вставляем приватные ключи (Не адреса) кошельков
- `python main.py`
